# Conversations — 세션 인수인계

## 역할

AI 에이전트 **세션 간 브릿지**입니다.
장기 지식은 `wiki/`로 승격하고, 여기는 임시 인수인계만 둡니다.

## 언제 쓰나

- 세션 종료 직전: 진행 중 작업, 미결정 사항, 다음 액션
- 멀티 에이전트 핸드오프: Claude → Codex 등 도구 전환 시

## 파일명 규칙

```
YYYY-MM-DD-주제-slug.md
```

## 권장 형식

```markdown
# 세션 인수인계 — YYYY-MM-DD

## 현재 상태
## 결정된 것
## 미결정 / 블로커
## 다음 액션
## 관련 wiki
- [[projects/...]]
```

## 주의

- conversations는 **저장 필터를 우회하지 않습니다**.
- 반복 참조될 내용은 `save`로 wiki에 올리세요.
