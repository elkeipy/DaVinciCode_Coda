# Save Prompt

```text
이번 작업 내용을 옵시디언에 저장해줘.

저장하기 전에 CLAUDE.md와 AGENTS.md의 저장 규칙을 확인하고, 아래 5가지 필터 중 최소 1개를 충족하는지 검증해줘.

1. 향후 실무에 반복 재사용될 데이터인가?
2. 다른 에이전트·동료가 이어받기 위해 반드시 읽어야 하는가?
3. 결정 근거·결정권자 추적이 필요한가?
4. 다시 시도하면 안 되는 리스크 정보인가?
5. 팀 공통 규칙·디자인 가이드인가?

저장할 가치가 있으면 적절한 wiki 카테고리를 판단해서 문서를 만들고, index.md와 log.md도 갱신해줘.

저장할 가치가 없으면 저장하지 말고 그 이유를 짧게 설명해줘.
```
